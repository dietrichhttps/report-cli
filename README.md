# Developer Performance Reports

CLI-утилита для анализа эффективности разработчиков по CSV-файлам с данными о закрытых задачах.

Поддерживает несколько отчётов, легко расширяется, покрыта тестами на ~99%, использует современный src-layout и устанавливается одной командой.

## Возможности

- Отчёт `performance` — средняя эффективность (performance) по позициям (по ТЗ)
- Отчёт `summary` — общая сводка: сколько сотрудников, команд, средний стаж, топ-навыки и т.д.
- Архитектура позволяет добавить новый отчёт за 5 минут

## Установка и запуск

```bash
# Клонируем и заходим в папку
git clone https://github.com/твой_ник/report-api.git
cd report-api

# Создаём виртуальное окружение (если ещё нет)
python3 -m venv .venv
source .venv/bin/activate

# Устанавливаем проект в режиме разработки
pip install -e .

# Готово! Теперь можно запускать:
report-api --files путь/к/файлам/*.csv --report performance
report-api --files data/*.csv --report summary
